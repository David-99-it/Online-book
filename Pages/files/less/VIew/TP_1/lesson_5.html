Глоссарий

В программировании, строковый тип (англ. string «нить, вереница») — тип данных, значениями которого является произвольная последовательность (строка) символов алфавита. Каждая переменная такого типа (строковая переменная) может быть представлена фиксированным количеством байтов либо иметь произвольную длину.

 

1.Представление строк в С++.

Для представления строковых данных на языке С++ используются переменные типа char и заголовочный файл <string.h>. Строка представляет собой массив символов, заканчивающийся нуль – символом, что означает конец этой строки.

Например. char str[10]=”Vista”;

  В этом примере под строку выделяется 10 байт, 5 из которых занято под символы строки, а шестой – под нуль - символ. Если строка при определении инициализируется, ее размерность можно опускать (компилятор сам выделит соответствующее количество байт)

char str[]= “Vista”; // выделено и заполнено 6 байт

2. Стандартные функции работы со строками.

1. strlen (str) – подсчитывает количество символов в строке.

Пример.

int x;

char str=”Пробная строка”;

x=strlen(str);

2. strcpy(str1, str2) – копирует одну строку в другую. В качестве исходной строки можно использовать тип данных const. Копирует из строки str2 в строку str1. И результат возвращает в str1.

Пример

char str1[15]=”Первая строка”;

char str2[15];

strcpy(str2, str1);

cout<<str2; //первая строка

3. strncpy(str1, str2, int n) – копирует указанное количество символов из одной строки в другую.

Функция копирует не более n символов из str2 в str1, и возвращает в str1.

пример:

strncpy (str2, str1, 5);

cout<<str2; // перва

4. strcat(str1, str2) - объединяет строк (конкатенация). Функция добавляет str2 к str1 и возвращает результат в str1.

char str1[15]=”Первая строка”;

char str3[50];

strncpy(str3, str1, 6);

strcat(str3,str1);

cout<<str3; // первая +первая строка

5. strncat(str1, str2, int n) – складывает одну строку с n символами другой.

Функция добавляет не более n символов из str2 к str1 и возвращает в str1.

strncpy(str3, str1, 6);

strncat(str3, str1,8);

cout<<str3; // первая первая с

6. strlwr(str) -преобразовывает символы верхнего регистра в символы нижнего регистра. Указывает на строки и возвращает указатель на эту строку.

Пример

strlwr(str1);

cout<<str1;

7. strupr(str1) – преобразует символы нижнего регистра в символы верхнего регистра. Указывает на строки и возвращает указатель на эту строку.

Пример

strupr(str2);

cout<<str2;

8. strrev(str) представляет символы заданной строки в обратном порядке (переворачивает) и возвращает указатель на эту строку.

Пример

strrev(str1);

cout<<str1;

9. strchr(str, c) – эта функция предназначена для поиска символов в строке. Эта функция возвращает указатель на первое вхождение символа с в строку str, а если его нет, то возвращается null.

пример

сhar *sptr;

sptr=strchr(str1,’p’);

cout<<sptr; // рока

10. strrchr(str, c) – ищет символ в строке. Определяет последнее вхождение символа в заданной строке и возвращает указатель на этот символ. Если этот символ не обнаружен в заданной строке, то возвращает 0.

Пример

char *sptr;

sptr=strrchr(str2,’o’);

cout<<sptr; // овая

 

Пример 1. Написать программу, которая производит со строками следующие действия:

1.Вычислить длину первой строчки.

2.Произвести копирование первой строки в третью.

3.Произвести объединение 2 и 3 строк.

#include <iostream>

#include <string.h>

int main()

{

setlocale(LC_ALL,"Russian");  

using namespace std;

char str1[40]="Это первая строка";

char str2[40]="Это вторая строка";

char str3[40]="Это третья строка";

int kol_symbol=0;

kol_symbol=strlen(str1); // вычисляем длину первой строчки

strcpy(str3,str1); // копируем первую строку в третью

strcat(str3,str2); //добавляем к третьей строчки вторую

cout<<str1<<endl;

cout<<str2<<endl;

cout<<str3<<endl;

cout<<kol_symbol<<endl;

 

system("PAUSE");

return 0;

}
