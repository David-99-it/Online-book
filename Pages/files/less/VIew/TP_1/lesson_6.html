Глоссарий

Функция в программировании — поименованный фрагмент программного кода (подпрограмма), к которому можно обратиться из другого места программы После выполнения функции, управление возвращается обратно в адрес возврата — точку программы, где данная функция была вызвана.

Перегрузка процедур и функций — возможность использования одноимённых подпрограмм: процедур или функций в языках программирования.

1.Функции

С помощью функции в С++ задачи могут быть разделены на более простые составляющие. Разделение программы на функции позволяет также избежать избыточности кода, поскольку функцию записывают один раз, а вызывать ее на выполнение можно сколько угодно раз из разных точек программы.

Функция – это именованный блок кода, невходящий непосредственно в основную программу, но вызываемый из нее при необходимости.

Функция - представляет собой набор связанных операторов, которые выполняют определенную задачу.

Если программе необходимо выполнить определенную задачу, то она вызывает соответствующую функцию. Каждая создаваемая в программах функция должна иметь уникальное имя. Функция простейшего типа не принимает никаких аргументов и не возвращает результат. Остальные функции могут принимать 1 или более аргументов и возвращать некоторый результат.

Аргумент (параметр) – это передаваемое функции значение, которое используется в блоке - кода функции. До использования функции ее необходимо объявить.

Объявлений может быть несколько, а определений одно.

Объявление функции содержит:

Тип возвращаемого результата Имя функции (аргумент 1, аргумент 2….n)

 

 

Указывает тип имя аргументы, которые

возв. рез-та передаются в эту функцию.

Определение функции – это ее объявление, в котором присутствует тело функции.

Например: int add_values(int, int);

  Функция начинает свое выполнение в момент вызова. Программы выполняют операторы функций посредством вызова функции. Для вызова функции программы просто обращаются к имени функции, за которым следуют круглые скобки, как показано ниже: function_name( );

После того как последний оператор функции завершен, выполнение программы продолжается с первого оператора следующего за вызовом функции.

Функции могут передавать любое количество аргументов, но возвращаемый результат всегда один.

- Выход из функции осуществляется по оператору return.

Пример 2. Написать функцию, которая осуществляет суммирование аргументов.

#include<iostream>

using namespace std;

int sum(int a, int b) // объявление функции

{

int c=a+b; // определение функции

return c;

}

int main()

{

setlocale(LC_ALL,"Russian");

int a,b,res;

cout<<"a=";

cin>>a;

cout<<"b=";

cin>>b;

res=sum(a,b);

cout<<"Сумма"<<res<<endl;

system("pause");

return 0;

}

Объявление функции можно отделить от определения, например в данной программе функцию sum можно объявить до функции main, а определить после:

#include<iostream>

using namespace std;

int sum(int a, int b); // объявление функции

int main()

{

setlocale(LC_ALL,"Russian");

int a,b,res;

cout<<"a=";

cin>>a;

cout<<"b=";

cin>>b;

res=sum(a,b);

cout<<"Сумма"<<res<<endl;

system("pause");

return 0;

}

int sum(int a, int b) // определение функции

{

int c=a+b;

return c;

}

2.Передача массивов в функции.

Пример 3. Написать программу, которая передает массивы в функцию show_array, и использует цикл for для вывода значений массивов.

#include <iostream>

using namespace std;

void show_array (int array [] , int number_of_elements)

{

int i;

for (i = 0; i < number_of_elements; i++) cout << array[i] << ' ';

cout << endl;

}

void main(void)

{

int little_numbers[5] ={1,2,3,4,5};

int big_numbers[3] = { 1000, 2000, 3000 };

show_array(little_numbers, 5);

show_array(big_numbers, 3);

system("PAUSE");

}

Результат программы:

1 2 3 4 5

1000 2000 3000

4.Перегрузка функций.

При определении функций в своих программах необходимо указать тип возвращаемого функцией значения, а также количество параметров и тип каждого из них.

Перегрузка функций позволяет программам определять несколько функций с одним и тем же именем и типом возвращаемого значения. В процессе компиляции C++ принимает во внимание количество аргументов, используемых каждой функцией, и затем вызывает именно требуемую функцию. Перегрузка функций достаточно удобна и упрощает работу программ.

Пример 5. Данная программа перегружает функцию с именем add_values.

#include <iostream>

using namespace std;

int add_values(int a,int b)

{

return(a + b);

}

int add_values (int a, int b, int c)

{

return(a + b + c);

}

void main(void)

{

cout << "2525 +4123 = " << add_values(2525, 4123) << endl;

cout << "105 + 209 + 700 = " << add_values(105, 209, 700) << endl;

system("PAUSE");

}

 

Результат:

2525 +4123 =6648

105 + 209 + 700 =1014

Перегрузка функций позволяет программам определять несколько функций с одним и тем же именем. Перегруженные функции должны возвращать значения одинакового типа, но могут отличаться количеством и типом параметров.

5.Значения параметров по умолчанию

C++ позволяет с помощью параметров передавать информацию в функции. Также C++ обеспечивает перегрузку функций, предусматривая определения, содержащие разное количество параметров или даже параметры разных типов. Кроме этого, в C++ при вызове функций можно опускать параметры. В таких случаях для опущенных параметров будут использоваться значения по умолчанию. Обеспечение значений по умолчанию для параметров упрощает возможность повторного использования функций.

Например:

имя функции (int size=12, float cost=19.95) //---->Значения по умолчанию

Пример 6. Данная программа показывает работу функции при использовании значения параметров по умолчанию.

#include <iostream>

#include <conio.h>

using namespace std;

void show_parameters (int a=1, int b=2, int c=3)

{

cout << "a=" << a << " b =" << b << " c= " << c << endl;

}

void main(void)

{

show_parameters();

show_parameters(1001);

show_parameters(1001, 2002);

show_parameters(1001, 2002, 3003);

getch();

}

Результат программы:

а =1 b =2 с =3

а =1001 b= 2 с= 3

а =1001 b= 2002 с =3

а =1001 b= 2002 с =3003
